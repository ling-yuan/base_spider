# 基于scrapy的多阶段式通用爬虫框架

## 前提环境

[python 官网](https://www.python.org/)

[make (windows)](https://gnuwin32.sourceforge.net/packages/make.htm)

## 安装

```bash
make
```

```bash
make install
```

在main.py中修改配置信息即可运行

> [!note]
>
> 配置文件信息模板参考: `universal_spider\template\config_template.py`
> 
> 栗子: `test_config.md`
>
